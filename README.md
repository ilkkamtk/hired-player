# hired-player

A simple PHP-based MP3 player for instruments.

## Setup

1. Place your MP3 files in the respective folders under `mp3/`:
   - `mp3/guitar/`
   - `mp3/keyboard/`
   - `mp3/vocals/`
   - `mp3/bass/`
   - `mp3/drums/` (special player coming soon)

2. Run the PHP server:

   ```
   php -S localhost:8000
   ```

3. Open `http://localhost:8000` in your browser.

## Features

- Tabs for each instrument
- Click to play MP3 files
- Simple player with play/pause and scrubbing
- Mobile-first responsive design

Note: Drum tab has placeholder; different functionality to be implemented later.
